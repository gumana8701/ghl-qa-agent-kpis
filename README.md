# GHL QA Agent KPIs — Case Settlement Now

Dashboard for QA scores, call attempt KPIs, and management alerts.

## Setup

1. Deploy to Vercel
2. Set environment variables:
   - `NEXT_PUBLIC_SUPABASE_URL=https://kgndqzrluavpnsomckzy.supabase.co`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY=<your anon key>`

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Supabase
